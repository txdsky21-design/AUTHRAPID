## Customization

The red theme colors are defined in CSS variables at the top of `styles.css`:

```css
:root {
    --primary-red: #DC2626;
    --primary-red-dark: #B91C1C;
    --primary-red-light: #EF4444;
    /* ... more variables */
}
```

You can easily customize colors by changing these variables.
